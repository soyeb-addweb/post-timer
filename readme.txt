Simple and easy-to-use WP plugin that displays a timer popup on selected posts, pages, or custom post types when a user adds or edits content.

== Description ==
Post Timer is a simple and easy-to-use WordPress plugin designed to display a timer popup on selected posts, pages, or custom post types when a user adds or edits content.

With this plugin, you can:
- Customize the timer popup color and placement (e.g., top-left, top-right, bottom-left, bottom-right, left, right).
- Show the timer popup on specific posts, pages, or custom post types during the add/edit process.

== Installation ==

**Installation Instructions & Configuration**  

1. Download the zip file and extract its contents. Upload the 'post_timer' folder to your plugins directory (`wp-content/plugins/`).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Post Timer** to configure the popup color, placement, and select which posts, pages, or custom post types will display the timer popup during post creation or editing.

== Frequently Asked Questions ==

= What is the use of this plugin? =
Post Timer displays a timer popup when users add or edit a post. This allows users to track how much time they spend on creating or editing posts.

= How do I change the popup settings? =
You can change the popup settings by going to **Dashboard > Settings > Post Timer**.
